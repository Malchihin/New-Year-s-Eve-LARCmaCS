Подключение робота:

1.      Открутить крышку робота

2.      Включить (короткое нажатие)

3.      Подождать, пока на экране появится IP робота

4.      Закрутить крышку

5.      LARCmaCS.exe –> Set up IP -> Ввести IP в ячейку с номером робота -> ОК

Проверка связи с роботом:

1.      Ввести IP робота в поле над Remote Control

2.      Нажать Remote Control (в этот момент управление роботом осуществляется с клавиатуры)

3.      Закрыть Remote Control

Возможные причины того, что алгоритм не работает:

1.      Нет подключения к сети Wi-Fi (сеть MAS2, пароль robotrobot)

2.      Открыт Remote Control

3.      Разряжен аккумулятор, обратитесь к организатору (индикация в виде одной красной лампочки сзади робота)

4.      Ctrl + S (убедитесь, что вы сохранили ваш код)

5.      Убедитесь, что в нижнем левом большом поле, в рамочке, в LARCmaCS’ e написано WORK

6.      Возможно, робот не распознается, его не видно в LARCmaCS, система видеозрения его не задетектила\ робота нет на поле

7.      Если в левом нижнем углу, в LARCmaCS’ e, появилась надпись «waiting for ssl connection», обратитесь к организатору

Скрипт main запускается каждый раз, когда приходит информация о роботе (~30 раз в секунду, камера работает ~30 fps).

Команда управления роботом:

Rule (номер робота, скорость по Х (|X| <= 100) , скорость по Y(|Y| <= 100), кикер ( 1 или 0) ,  |угловая скорость| <= 100 , 0)

Выполняется только последняя переданная команда на робота.

Ex: Rule (4, 30, 30, 0, 0) – робот №4 едет по диагонали

 

 

Информация о роботе, которую можно получить из консоли MatLab:

1.      Команда: «RP.Blue(номер робота)»

         Вывод:

I - распознает ли робота камера

                   Z – (X, Y)

                   X – координата по х(последняя переданная камерой)

                   Y – координата по у(последняя переданная камерой)

                   Ang – угол между осью x камеры и осью у робота (у робота направлен туда же, куда «смотрит» робот)

             v    - Скорость движения робота

             u – управляющее воздействие

            Nrul  =  -1        - Номер исходящего управления

            rul: [1x1 struct]   - Исходящее управление

          KickAng - Направление удара робота (на данный момент неизменяемое)

2.      Команда: «Blues»

         Вывод: массив строк для 12 роботов, номер строки соответствует номеру робота (отсчет ведется с 1):

1 столбец – присутствует ли робот на поле

2 столбец – координата Х (последняя переданная камерой)

3 столбец – координата У(последняя переданная камерой)

4 столбец – угол

 

3.      Команда: «RP.Ball»

         Вывод:

I - распознает ли мяч камера

                   X – координата по х

                   Y – координата по у

                   Z – (X, Y)

                   Ang – ?

             v  - Скорость движения мяча

             id = 100
